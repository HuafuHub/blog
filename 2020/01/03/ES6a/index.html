二、变量声明

1.var 特点

  • 变量会声明提升，允许先使用后声明
  • 语法松散，重复声明变量不会报错，会覆盖
  • 全局变量会挂载 window
  • 函数局部作用域容易产生闭包

2.let 特点

  • 不会变量声明提升，需先声明后使用
  • 不得重复声明
  • 不会挂载全局 window 属性
  • 与大括号{}配合使用形成块级作用域
    临时死区
  • 称霸当前块级作用域（只要有就必须使用当前块内的变量）
  • 而且声明之前不允许使用，也不使用外部同名变量
  • 具有识别块级作用域的能力

3.const 特点

  • 声明常量，常量名建议大写
  • 在声明时就必须初始化值
  • 存储常量的空间里面的值不能发生改变
  • 如若值是原始类型，则无法改变
  • 如若值是引用类型，则地址无法改变
  • 具有识别块级作用域的能力

四、箭头函数

1.存储

  • 必须有空间存储，变量、数组、对象属性等

2.参数

  • 当只有一个形参的时候，可以胜率圆括号
  • 多个形参名不能相同（报错）

4.返回值

  • 可以省略 return 关键字，省略大括号，直接写在箭头后

5.arguments/this 指向

  • arguments 由定义时外层最接近的非箭头函数的 arguments 决定
  • this 由定义时外层最接近的非箭头函数的 this 决定
  • 全局定义的箭头函数的 arguments 未定义
  • 全局定义的箭头函数的 this 指向 window

七、数据存储

（一）Set

1.创建 new Set(data)

  • 产生一个类似数组的机构，里面的值唯一
  • data 具备迭代接口的数据
  • Set 只存储唯一 value，不存储 key

2.原型方法

  • add(value)添加数据
  • delete(value)删除数据==value 必须与存储的一致否则删除不了（字面量形式就删除不了，用变量接受，存储变量）==
  • clear()清空数据
  • has(value)检测数据 true/false

3.遍历

  • for…of 遍历迭代接口的数据
  • forEach 方法

4.转换

  • Array.form(Set)
  • […Set]

5.应用

  • 去重
  • 差集
  • 交集
  • 并集

（二）Map

1.创建 new Map([[key, value]])

  • “做字典”
  • data 具备迭代接口的数据
  • Map 存储唯一 key-value 键值对
  • key 可以是任何形式

2.原型方法

  • Map(key, value)添加数据
  • get(key)获取数据==key 必须与存储的一致否则获取不了（字面量形式就获取不了，用变量接受，存储变量）==
  • delete(key)删除数据
  • keys()获取所有的键
  • has(key)检测数据

3.遍历

  • for…of 遍历迭代接口的数据
  • forEach 方法

4.模拟（链接列表/*桶、hash 算法）
