3.图片

（1）src属性（资源路径）

  • 路径引入资源
  • data URL（data:MIME,base64,数据）
    • 优点：减少浏览器的请求；动态数据
    • 缺点：增加资源体积（传输的内容体积会增加4/3）；不利于浏览器缓存；
    • 应用：个体体积较小，不适宜使用雪碧图；图片由其它语言动态生成，且图片较小

（2）alt属性（资源失效替代方案）

  • 链接失效时，宽高也会失效（设置成块盒或行块盒）

（3）与a元素联用（图片实现跳转链接）

  • 行盒里出现行块盒或行盒，自身的高度只跟字体大小有关（不会取决于内部的行盒或行块盒）

（4）与map元素联用（map元素可以实现指定图片不同区域跳转对应链接）

（5）与figure元素联用（figure元素语义化更好的将相关内容包裹起来）

（6）常见问题

  • 图片区域坍塌

    • 链接失效时，行盒宽高也会失效
    • 图片设置成块盒（行块盒）指定宽高

  • 超链接包裹高度坍塌

    • 行盒包裹行盒（行块盒），自身的高度只跟字体大小有关，不会自适应内部的行盒（行块盒）
    • 超链接设置成块盒（行块盒）

  • 块盒包裹底部白边

    • 图片的基线在下外边距位置
    • 块盒内部有行盒（行块盒），也会生成参考线（font-family、font-size、line-height）
    • 不生成行框：某元素内没有任何行盒（行块盒）；某元素字体大小为0
    • 块盒字体大小设置成0（有副作用）
    • 图片设置成块盒

5.表单

（1）input元素（输入框）

  • type属性：输入框类型
    • text文本输入框
    • password 密码输入框
    • date 日期选择框（兼容性问题）
    • search 搜索框（兼容性问题）
    • range 滑块（兼容性问题）
    • color 颜色选择（兼容性问题）
    • number 数字输入框（兼容性问题）
    • CheckBox 多选框
    • radio 单选框
    • file 文件选择器
    • reset 重置按钮
    • button 点击按钮

  • name限定（同一题）

（2）select元素（下拉列表）

（4）button元素（按钮）

（5）表单附属元素

4）fieldset元素（表单分组）
  • legend 分组标题（子元素）

  • readonly属性 只读

（6）美化表单元素

1）聚焦 :focus

10.canvas

1)初始化

  • 创建画布
  • html 中 canvas 标签指定 3 个属性 id 标识、width 画布宽度、height 画布高度
  • 获取画布
  • var canvas = document.getElementById()
  • 获取画笔
  • var ctx = canvas.getContext(“2d”)

2)线性路径

  • 创建路径
  • ctx.beginPath()
  • 画笔粗细
  • ctx.lineWidth = num
  • 线端样式
  • ctx.lineCap = “button”/“square”/“round”
  • ctx.lineJoin = “miter”/“bevel”/“round”
  • 画笔定位
  • ctx.moveTo(x, y)
  • 画线
  • ctx.lineTo(x, y)
  • 闭合路径
  • ctx.closePath()
  • 描边
  • ctx.strokeStyle = “red”
  • ctx.stroke()
  • 填充
  • ctx.fillStyle = “red”
  • ctx.fill()
  • 擦除
  • ctx.clearRect(x,y,w,h)

3)基础图形路径

<1>rect 矩形
  • ctx.rect(x, y, w, h)
<2> arc 弧形（圆形）
  • ctx.arc(ox, oy, r, begin, end, dirction)
<3> arcTo 圆角
  • ctx.arcTo(x, y, dx, dy, r)

4)贝塞尔曲线

<1> 二次贝塞尔
  • ctx.quadraticCurveTo(x1, y1, x, y)
  • 从上一点开始绘制一条二次曲线，到(x, y)为止，并且以(x1, y1)作为控制点
<2> 三次贝塞尔
  • ctx.bezierCurveTo(x1, y1, x2, y2, x, y)
  • 从上一个点开始绘制一条曲线，到(x, y)为止，并且以(x1, y1)和(x2, y2)为控制点

5)坐标系控制

  • ==全局控制==
<1> translate 平移坐标系原点
  • ctx.translate(x, y)
  • 原坐标系原点在画布左上角点
<2> rotate 旋转路径
  • ctx.rotate(Math.PI)
  • 相对于坐标原点进行旋转
<3> scale 路径缩放
  • ctx.scale(x, y)
  • 每个点的位置进行相应倍数的缩放
<4> save/restore 保存、恢复坐标系状态数据
  • ctx.save()
  • 保存坐标系状态（平移、旋转、缩放）
  • ctx.restore()
  • 恢复保存时的坐标系状态

6)特殊填充效果

  • 从坐标系原点开始填充
<1> createPattern 纹理
  • ctx.createPattern(img)
<2> createLinearGradient 线性渐变
  • ctx.createLinearGradient(x1,y1,x2,y2)
  • bg.addColorStop(0, “red”)
  • bg.addColorStop(1, “blue”)
<3> createRedialGradient 径向渐变
  • ctx.createRedialGradient(x1,y1,r1, x2,y2,r2)
  • bg.addColorStop(0, “red”)
  • bg.addColorStop(1, “blue”)

7）阴影

  • 阴影颜色
  • ctx.shadowColor = color
  • 模糊半径
  • ctx.shadowBlur = num
  • 偏移量
  • ctx.shadowOffsetX = num
  • ctx.shadowOffsetY = num
